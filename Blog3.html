VARIABLES & DECLARATIONS : Firstly variables are containers that store data. The word "variable" refers to meaning "changable".
These statements are mainly divided into three types
VAR , LET , CONST.

VAR : This statement declares function-scoped or globally-scoped(file-scoped) variable available before declaration and can be changed inside the scope

LET: This statement declares block-scoped variable, where it can be accessed within a block of code within braces mainly and will available after it's declaration only hence can be changed inside the scope.

CONST : It's also a block-scoped statement , that can never be changed once declared. variable available only affter it's declaration.
